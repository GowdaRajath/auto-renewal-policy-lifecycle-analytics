# Synthetic Auto-Renewal Dataset

This dataset is **synthetic and anonymised**, created solely for analytical demonstration purposes.

It does not contain:
- Real customer data
- Employer-specific information
- Personally identifiable information (PII)
- Proprietary system logic

The dataset is designed to replicate realistic auto-renewal complaint scenarios observed within regulated insurance environments, enabling timeline-based analysis of customer awareness, communication timing, escalation behaviour, and complaint preventability.

All dates, identifiers, and outcomes are fictional.
